# ConcLand Advanced - 高度シミュレーションアーキテクチャ

## 🎯 プロジェクト概要

戦後日本復興（1945-1970年）をテーマにした高度都市建設シミュレーション。
焼け野原から近代都市への発展を、歴史的文脈と共に体験できるゲーム。

## 🕰️ 時代システム (1945-1970)

### 時代区分
1. **終戦直後** (1945-1950)
   - 焼け野原からの復興
   - 基本インフラの再建
   - 食糧危機との戦い

2. **朝鮮戦争特需** (1950-1955)
   - 経済復興の契機
   - 工業生産の急成長
   - 都市化の始まり

3. **高度成長期前期** (1955-1965)
   - 神武景気、岩戸景気
   - 大量生産・大量消費
   - インフラ整備本格化

4. **東京オリンピック時代** (1965-1970)
   - 国際社会復帰
   - 新幹線・高速道路
   - 現代都市の完成

### 時代別解禁要素

#### 1945-1950年
- **建物**: バラック住宅、個人商店、派出所、小学校
- **資源**: 米、薪炭、鉄くず
- **技術**: 基本建設、農業復興

#### 1950-1955年
- **建物**: 市営団地、商店街、小工場、診療所
- **資源**: 石炭、鉄鋼、繊維
- **技術**: 軽工業、基礎化学

#### 1955-1965年
- **建物**: 高層団地、百貨店、総合病院、国鉄駅
- **資源**: 石油、プラスチック、電子部品
- **技術**: 重工業、家電産業

#### 1965-1970年
- **建物**: 高速道路、テレビ塔、原子力発電所、空港
- **資源**: 原子力、半導体、自動車
- **技術**: 原子力、宇宙開発

## 💰 資源経済システム

### 基本資源 (6種類)
1. **米** - 食糧の基本、人口に直結
2. **鉄・鋼鉄** - 建設・工業の基礎
3. **木材** - 建設資材、燃料
4. **石炭** - エネルギー源（～1960年代）
5. **石油** - 近代エネルギー（1955年～）
6. **電力** - 全産業に必要

### 経済メカニズム
- **月次精算**: 収入・支出の月末計算
- **貿易システム**: 資源の輸出入
- **価格変動**: 需給バランスによる価格変化
- **特需効果**: 朝鮮戦争特需などのブースト

## 📅 歴史的イベントシステム

### 主要イベント
1. **1950年**: 朝鮮戦争勃発 → 特需景気
2. **1953年**: テレビ放送開始 → 家電ブーム
3. **1956年**: 経済白書「もはや戦後ではない」
4. **1958年**: 東京タワー完成
5. **1964年**: 東京オリンピック開催
6. **1968年**: GNP世界第2位達成

### イベント効果
- **経済効果**: 資源価格変動、建設需要増
- **技術効果**: 新建物・技術の解禁
- **社会効果**: 人口動態、幸福度の変化

## 🌸 季節・災害システム

### 季節変化
- **春**: 桜、入学・就職シーズン
- **夏**: 祭り、電力需要増
- **秋**: 収穫、観光シーズン
- **冬**: 暖房需要、雪害

### 災害システム
- **地震**: 関東大震災規模の被害
- **台風**: 伊勢湾台風など
- **火災**: 木造建築の脆弱性
- **水害**: 河川氾濫、高潮被害

## 🔬 技術ツリーシステム

### 技術分野
1. **建設技術**: 建物効率・耐久性向上
2. **工業技術**: 生産効率向上
3. **エネルギー技術**: 発電効率・新エネルギー
4. **交通技術**: 道路・鉄道・航空
5. **通信技術**: 電話・テレビ・ラジオ
6. **医療技術**: 病院効率・公衆衛生

### 研究システム
- **研究機関**: 大学、研究所で技術開発
- **技術導入**: 海外技術の導入コスト
- **実用化**: 技術から実際の建物・システムへ

## 📊 評価システム

### 評価要素
1. **人口規模**: 最終人口数
2. **経済規模**: GDP、一人当たり所得
3. **都市機能**: インフラ整備度
4. **技術水準**: 技術開発達成度
5. **市民幸福度**: 生活の質指標
6. **歴史再現度**: 史実との整合性

### ランク評価
- **S評価**: 奇跡の復興 (全要素90%以上)
- **A評価**: 模範的復興 (全要素80%以上)
- **B評価**: 順調な復興 (全要素70%以上)
- **C評価**: 平均的復興 (全要素60%以上)
- **D評価**: 困難な復興 (全要素50%以上)
- **E評価**: 停滞する復興 (全要素50%未満)

## 🎮 ゲームモード

### 1. ストーリーモード
- 1945年から段階的に時代が進行
- 歴史イベントが自動発生
- チュートリアル機能付き

### 2. フリープレイモード
- 好きな年代からスタート
- イベント発生をカスタマイズ
- チート機能・デバッグ機能

### 3. チャレンジモード
- 特定条件での達成目標
- 災害多発、資源制限など
- オンラインランキング

### 4. マルチプレイモード
- 複数プレイヤーで地域分担
- 貿易・協力システム
- 競争・協調の選択
