# 簡素化された操作ガイド

##  基本操作

### カーソル移動
- **矢印キー**: ⬆️⬇️⬅️➡️
- **または K/J/H/L キー**（Vimライク）

### アクション
- **スペースキー** または **Zキー**: 建物を配置
- **Xキー**: 建物を削除
- **Hキー**: ヘルプ表示/非表示（いつでも確認可能！）

---

##  ツール選択（QWERTY配列で直感的に）

### 基本ゾーン
| キー | ツール | 説明 |
|------|--------|------|
| **Q** | 🏠 住宅 | 住宅ゾーンを配置 |
| **W** | 🏢 商業 | 商業ゾーンを配置 |
| **E** | 🏭 工業 | 工業ゾーンを配置 |

### インフラ
| キー | ツール | 説明 |
|------|--------|------|
| **R** | 🛣️ 道路 | 道路を敷設 |
| **T** | 🚆 鉄道 | 鉄道・駅（繰り返しで切替） |
| **U** | ⚡ 電線 | 電線を敷設 |

| **Y** | 🌳 公園 | 公園を配置（繰り返しで切替） |

| **O** | 🚢 港湾 | 港湾施設（繰り返しで種類切替） |

##  便利な機能

### 表示切替
- **Vキー**: 表示モードを循環切替
  - 通常 → 汚染 → 地価 → 電力 → 交通 → 通常...

### 詳細UIパネル
- **S**: 統計パネル（人口・雇用など）
- **E**: 経済パネル（資源・市場など）
- **T**: 交通パネル（渋滞・路線など）
- **D**: 災害パネル（災害・緊急サービスなど）
- **P**: 政策パネル（税金・政策など）

### ツールパレット
- **Bキー**: 全ツール一覧を表示
  - 左右キーで選択
  - スペースで決定
  - ESCでキャンセル

##  はじめてのプレイガイド

### ステップ1: 道路を作る
1. **Rキー** を押して「道路」を選択
2. **矢印キー** でカーソルを移動
3. **スペースキー** で道路を配置

### ステップ2: 住宅を作る
1. **Qキー** を押して「住宅」を選択
2. 道路の隣に **スペースキー** で配置
3. これで人々が住み始めます

### ステップ3: 電力を引く
1. **Iキー** を押して「発電所」を選択
2. 資金が足りない場合は待機
3. **Uキー** で電線を選択し、発電所から住宅へ引く

### ステップ4: 発展を見守る
- 時間が経つと住宅が発展します
- 人口が増えると税収入が増えます
- さらに多くの施設を建てられるように

---

##  トラブルシューティング

### 資金が足りない
- **解決策**: 住宅の成長を待って税収入を増やす
- **ヒント**: まずは道路と住宅に集中しよう

### 人口が増えない
- **確認ポイント**:
  - 道路に隣接しているか？
  - 電力が届いているか？
  - 汚染がひどくないか？
- **解決策**: 公園を配置して地価を上げる

### 操作がわからなくなった
- **Hキー** を押す！いつでも操作ガイドが表示されます
- **ESCキー** でUIパネルを閉じることができます

##  上級テクニック

### 効率的な都市計画
1. **住宅 → 商業 → 工業** のバランスが重要
2. 公園や警察署を設置すると周辺の地価が上がる
3. 工業は汚染発生源なので、住宅から離して配置

### 節約テクニック
- 最初は道路と住宅に集中
- 電力網を早めに整備
- 公共サービスは資金に余裕ができてから

### 情報活用
- **Vキー** で表示を切り替えて都市の状態を確認
- 汚染が少ないほど地価が上がる
- 交通が滞っていると商業が発展しない

---

**最後に**: 操作に困ったら **Hキー** を押してください！
