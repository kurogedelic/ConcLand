Controls

  • WASD - Move cursor (Arrow keys may not work in browser)
  • TAB - Cycle focus (Game → Palette → View)
  • 1-9,0,-,= - Select buildings
  • F1-F5 - Switch panels | O Save | I Load
  • Space/Z - Build | ESC - Return to game

Note: Click game window before playing
